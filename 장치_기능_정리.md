# 워터마스터 기능 정리

## 1. 장치 개요

이 장치는 다음 기능을 수행합니다.

- 화면에 수위, 전류, 유량, 날짜/시간, 통신 상태를 표시
- RF 통신으로 원격 센서 데이터를 주기적으로 수신
- 우물 수위 아날로그 입력을 읽어 표시 및 보호 제어 수행
- 릴레이 3종을 이용해 패널 출력, 모터, 알람 출력 제어
- Wi-Fi + MQTT를 통해 상태를 서버로 전송하고 펌프 원격 제어 수신
- BLE로 설정값 조회/수정 및 RTC 시간 동기화 지원
- 부팅 시 특정 버튼 입력으로 BLE OTA 모드 진입
- 자정 자동 재부팅 및 수동 리셋 버튼 지원

## 2. 화면 표시 항목

메인 화면(Page 0)에는 아래 정보가 표시됩니다.

- RF 채널: `CH-0` ~ `CH-9`
- LTE/Wi-Fi 네트워크 상태
- 앱/탱크 버전 정보
- 탱크 수위
- 우물 수위
- 정지 수위, 운전 수위, 알람 수위
- 모터 전류
- 유량값
- 설치 날짜/현재 시각
- 마력 정보
- TX/RX 통신 아이콘
- RF 안테나 아이콘
- 센서 타입 아이콘
- 펌프 아이콘
- 태양광 아이콘
- 배터리 아이콘
- 오류 카운트 또는 정상 표시

## 3. 주요 동작 기능

### 3.1 RF 수신 및 수위 표시

- 약 2초 주기로 RF 통신
- 응답이 오면 센서 타입(`Q`, `P`), 센서값, 배터리, 태양광 상태를 파싱
- 수신 성공 시 탱크 수위, 센서 타입, 배터리, 태양광 아이콘 갱신
- 수신 실패가 누적되면 오류 카운트 증가
- 60초 이상 정상 수신이 없으면 통신 실패 상태로 판단

### 3.2 유량 조회

- 유량계 설치 옵션이 켜진 경우 유량 요청
- 유량 응답이 정상이면 유량값 표시
- 응답이 없거나 형식이 맞지 않으면 `NO-ANS` 표시
- 유량계 미설치 시 유량 표시를 비움

### 3.3 전류 측정

- 약 1초 주기로 모터 전류 측정
- 설정된 제로 오프셋이 있으면 보정 후 표시
- 메인 페이지에서 조합 버튼으로 전류 영점 보정 가능

### 3.4 우물 수위 측정 및 보호

- ADC로 우물 수위 센서 전압을 읽어 미터(m) 단위로 환산
- 센서 미연결 시 `xxxx` 표시
- 유효 범위 미만이면 `----` 표시
- 우물 수위가 너무 낮으면 모터 운전을 차단하는 lockout 동작 수행
- 우물 수위가 회복 기준 이상이 되면 lockout 해제

### 3.5 펌프 자동 운전

- 탱크 수위를 기준으로 펌프 ON/OFF 판단
- 초기 상태에서는 정지 수위 이하일 때 운전 시작
- 운전 중에는 운전 수위 이하에서 유지, 정지 수위 이상이면 정지
- 통신 실패 시 펌프는 강제 정지

### 3.6 릴레이 출력 제어

- 릴레이1: 패널 지상/지하
- 릴레이2: 모터 출력 제어
- 릴레이3: 알람 출력 제어

릴레이3 알람 조건은 다음과 같습니다.

- RF 통신 실패
- 우물 수위 lockout 발생
- 탱크 저수위 알람
- 압력 센서 사용 시 고수위 알람

### 3.7 RTC 및 자동 재부팅

- RTC시간을 화면에 표시
- BLE로 휴대폰 시간을 받아 RTC 동기화 가능
- 매일 00:00에 1회 자동 재부팅 수행

### 3.8 BLE 설정 기능

- BLE 장치명: `GSWater`
- BLE UART 방식으로 설정 명령 수신
- 설정값 단건 또는 다건 수정 지원
- 현재 설정 전체 조회 지원
- 휴대폰 시간으로 RTC 동기화 지원

### 3.9 Wi-Fi / MQTT 기능

- 설정된 SSID/PASS로 Wi-Fi 접속
- 설정된 서버 주소로 MQTT 연결
- 런타임 상태를 주기적으로 MQTT 발행
- 메타 정보는 변경 시 또는 장시간 경과 시 재발행

전송 데이터에는 아래 정보가 포함됩니다.

- 지역 코드
- 장치 코드
- 수위
- 우물 수위
- 전류
- 유량
- 펌프 상태
- 알람 상태
- 배터리/태양광 상태
- 마력
- 설치일
- 전화번호 목록

## 4. 버튼 기능
- `btn_up`
- `btn_down`
- `btn_left`
- `btn_right`
- `btn_set`
- `btn_enter`
- `btn_esc`
- `btn_reset`

### 4.1 공통 버튼 기능

- `LEFT`: 이전 페이지로 이동
- `RIGHT`: 다음 페이지로 이동
- `RESET`: 즉시 시스템 재부팅

주의:

- 현재 코드 기준으로 `ESC` 단독 기능은 구현되어 있지 않습니다.

### 4.2 메인 페이지(Page 0) 버튼 기능

- `SET + UP`: RF 채널 증가
- `SET + DOWN`: RF 채널 감소
- `SET + ENTER`: 전류 영점 보정 실행
- `ENTER + SET`: 전류 영점 보정 실행
- `LEFT / RIGHT`: 페이지 이동

### 4.3 설정 페이지(Page 1~3) 버튼 기능

설정 페이지에서는 `SET` 버튼으로 설정 모드 진입/종료를 수행합니다.

- `SET`: 설정 모드 진입 또는 종료
- `LEFT`: 이전 항목 이동 또는 커서 왼쪽 이동
- `RIGHT`: 다음 항목 이동 또는 커서 오른쪽 이동
- `ENTER`: 다음 설정 항목으로 이동
- `UP`: 현재 항목 값 증가 또는 토글
- `DOWN`: 현재 항목 값 감소 또는 토글

설정 모드 세부 동작:

- 텍스트 항목: 문자 순환 편집
- 수위 항목: 10 단위 증가/감소
- IP 항목: 커서 위치 숫자 1자리씩 증감
- 아이콘 항목: ON/OFF 토글
- 60초 동안 입력이 없으면 설정 모드 종료 후 메인 페이지로 복귀

### 4.4 부팅 시 버튼 기능

- 부팅 직후 `SET` 버튼이 눌려 있으면 BLE OTA 모드 진입
- 연속 샘플 중 2회 이상 감지되면 OTA 모드로 판단
